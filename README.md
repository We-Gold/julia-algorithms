# Julia Algorithms
Learning Julia by implementing various common and uncommon algorithms. 

Note: I am by no means an expert, and I cannot promise that my code is great quality. If you want to improve my implementations, feel free!

## Algorithms Implemented

### Sorting Algorithms
- Bubble Sort
- Merge Sort
- Selection Sort
- Counting Sort
- Quick Sort

### Tree Algorithms
- Binary Tree
    - Preorder (DFS)
    - Inorder (DFS)
    - Postorder (DFS)
    - Level Order (BFS)

### Conversions
- Binary To Decimal

### Miscellaneous Algorithms
- Minimum Coins
- Fibonacci
- Knapsack Problem

*And many more to come*
